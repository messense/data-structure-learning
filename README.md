# 数据结构与算法

数据结构学习，作业代码

## 下载

    git clone git://github.com/messense/data-structure-learning.git

## 编译

    cd data-structure-learning && make build

## 清理
    cd data-structure-learning && make clean

## 测试

首先编译，然后
    cd data-structure-learning/bin
依次执行生成的可执行文件进行测试，如执行helloworld：
    ./helloworld
